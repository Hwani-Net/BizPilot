# BizPilot 🚗 Presentation Script
> 소상공인용 AI 전화 비서 & 올인원 백오피스

---

## 🚀 1. Project Overview (프로젝트 개요)

**"사장님, 전화 받느라 정비 흐름 끊기시죠?"**
BizPilot은 1인 자동차 정비소 사장님을 위해 **AI 전화 응대, 실시간 예약, 자동 장부, 재방문 마케팅(RCE)**을 하나로 통합한 AI 백오피스 솔루션입니다.

---

## 🧐 2. Problem (문제점)

1.  **전화 업무 과부하**: 작업 중 계속 걸려오는 전화(예약 문의, 가격 질문)로 인해 정비 효율 저하. 전화를 못 받으면 매출 기회 손실.
2.  **낙후된 관리 방식**: 종이 장부나 엑셀에 의존하여 고객 이력 관리가 안 됨.
3.  **수동적인 영업**: 고객이 고장 나서 찾아올 때까지 기다리기만 함. 정기 점검 시기를 놓친 고객을 놓침.

---

## 💡 3. Solution (핵결책)

### ① AI Phone Agent (AI 전화 비서)
- **OpenAI Realtime API + Twilio** 기반으로 사람처럼 자연스러운 대화 가능.
- 고객의 용무(예약, 가격 문의)를 파악하고, **실시간으로 예약 가능 시간을 조회하여 확정**까지 완료.
- 사장님은 작업에만 집중하고, 결과는 알림으로 확인.

### ② Revenue Continuity Engine (RCE)
- 차량별 연평균 주행거리를 분석하여 **소모품 교체 시기 예측**.
- "엔진오일 교환하신 지 6개월 지났네요" 같은 **개인화된 재방문 유도 문자** 자동 발송.
- 비수기 매출 방어의 핵심 엔진.

### ③ Automated Ledger (자동 장부)
- 부품 영수증을 **사진 찍기만 하면 OCR로 자동 입력**.
- 예약 내역과 연동되어 예상 매출과 실매출을 한눈에 파악.

## 🛠️ 4. Tech Stack (기술 스택)

| 분류 | 기술 | 역할 |
|---|---|---|
| **AI / Voice** | **OpenAI Realtime API** | 실시간 음성 대화, Function Calling (예약/조회) |
| | **Twilio** | 전화망 연결 (PSTN), Media Stream WebSocket |
| **Backend** | **Fastify (Node.js)** | 경량화된 고성능 서버, WebSocket 처리 |
| | **SQLite (better-sqlite3)** | 로컬 파일 기반 DB (빠른 프로토타이핑 & 배포 용이) |
| **Frontend** | **React 19 + Vite** | 반응형 SPA 대시보드 |
| | **TailwindCSS** | 모던한 유틸리티 퍼스트 스타일링 |
| **Deployment** | **Render / Cloudflare** | 서버 및 클라이언트 호스팅 |

## 🌟 5. Business Value (기대 효과)

- **매출 증대**: 놓치는 전화 0건 도전 + RCE로 재방문율 30% 증가 목표.
- **비용 절감**: 별도의 경리 직원 없이 AI가 24시간 응대.
- **삶의 질 향상**: 사장님이 "정비 기술"에만 집중할 수 있는 환경 제공.
