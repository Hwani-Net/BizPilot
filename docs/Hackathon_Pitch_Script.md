# 🎤 BizPilot 해커톤 발표 대본 및 시연 시나리오 (5분)

🟢 **라이브 데모 (Vercel)**: [https://bizpilot-jdah7xqrg-stayicon-gmailcoms-projects.vercel.app](https://bizpilot-jdah7xqrg-stayicon-gmailcoms-projects.vercel.app)
🟢 **백엔드 API (Render)**: [https://bizpilot-server.onrender.com](https://bizpilot-server.onrender.com)

## 1. 도입 (Intro & Hook) - 1분
**"안녕하세요, 소상공인의 잃어버린 매출을 찾아주는 AI 백오피스, 비즈파일럿(BizPilot) 팀입니다."**

*   **문제 제기**: 전국 10만 개가 넘는 카센터, 미용실 등 오프라인 매장의 가장 큰 고민은 무엇일까요? 바로 '전화 응대로 인한 업무 마비'와 '단골 고객의 이탈'입니다. 정비 중 걸려오는 전화를 받지 못해 놓치는 콜이 하루 평균 X건, 제때 교환 주기를 안내하지 않아 이탈하는 고객이 X%에 달합니다.
*   **솔루션 제시**: 저희는 이 문제를 AI로 해결합니다. 비즈파일럿은 매장 전화를 대신 받고, 고객의 방문 주기를 분석하여 자동으로 재방문을 유도하는 **'매출 연속성 엔진(RCE)'**을 탑재한 올인원 SaaS입니다.

## 2. 라이브 데모 시연 (Live Demo) - 2.5분

**(1) AI 전화 에이전트 (Calls 탭 시연)**
*   **시연**: `Calls` 메뉴 이동 -> `[시뮬레이션 시작]` 클릭
*   **대본**: "먼저, 정비사가 바쁠 때 전화를 대신 받는 AI 에이전트입니다. 고객이 전화를 걸면 AI가 차량 번호와 과거 정비 이력을 즉시 식별하여 맞춤형으로 응대합니다. 보시다시피 실시간으로 대화 내용이 텍스트로 트랜스크립트되며, 긴급한 예약 건은 즉각 대시보드로 인입됩니다."

**(2) RCE 캠페인 매니저 (Rce 탭 시연)**
*   **시연**: `Rce` 메뉴 이동 -> 리스트와 경고 배너 확인 -> `[원클릭 캠페인 발송]` 클릭 -> 폭죽 애니메이션 & Toast 확인
*   **대본**: "비즈파일럿의 핵심인 RCE(Revenue Continuity Engine) 기능입니다. 고객의 주행 거리와 마지막 엔진오일 교환 시기를 AI가 계산하여, 교환이 임박한 고객을 자동으로 필터링합니다. 현재 2대의 차량이 이탈 위험, 즉 '기회 감지' 상태네요. 
이때 '원클릭 캠페인 발송'을 누르면... AI가 고객별 최적화된 문구를 생성해 즉각 카카오톡/SMS를 발송합니다. 수기 관리로는 절대 불가능한 타이밍 마케팅을 버튼 하나로 끝냈습니다."

**(3) 실시간 수익 대시보드 (Dashboard 탭 시연)**
*   **시연**: `Dashboard` 메뉴 이동 -> 차트 마우스 오버
*   **대본**: "이러한 활동의 결과는 실시간 대시보드에 즉각 반영됩니다. RCE 캠페인을 통해 회수된 예약과 현장 결제 데이터가 통합되어, 이번 주 매출 추이와 AI 에이전트가 방어한 놓친 콜 수익을 직관적으로 확인할 수 있습니다."

## 3. 비즈니스 모델 및 시장성 (Business & Market) - 1분
*   **수익 모델**: 
    - Basic (월 3만원): RCE 자동화 + 고객 관리
    - Pro (월 10만원): AI 전화 응대 무제한 + 맞춤형 AI 모델 도입
*   **시장성**: 카센터로 시작하지만, 예약과 주기적인 재방문이 필요한 네일샵, 미용실, 동물병원 등 모든 **O2O(Online to Offline) 서비스업 백오피스**로 완벽하게 횡전개(Pivot)가 가능합니다. 

## 4. 아키텍처 및 마무리 (Outro) - 0.5분
*   **기술 스택**: Frontend는 React/Vite/Tailwind 기반의 반응형 Glassmorphism UI를 적용해 프리미엄 경험을 제공하며, Backend는 Node.js와 Supabase를 활용하여 실시간 데이터 동기화를 구현했습니다. 또한 심도있는 AI 기능은 최신 Gemini/Claude 모델을 멀티모달로 연결했습니다.
*   **마무리**: "소상공인의 가장 든든한 AI 부조종사. 지금까지 비즈파일럿(BizPilot)이었습니다. 감사합니다!"

### 💡 발표 가점 포인트 (심사위원 어필용)
1. **Glassmorphism 기반의 유려한 UI/UX**: 단순한 B2B 어드민을 넘어선 엔터프라이즈급 디자인. (마이크로 애니메이션 강조)
2. **실전성**: 실제 DB 데이터 기반 차트 렌더링, 현실적인 AI Call Mockup 등 '당장 팔 수 있을 것 같은' 완성도.
3. **독창적 엔진(RCE)**: 흔한 CRM이 아니라, 행동을 유도하는 'Revenue Continuity Engine'이라는 워딩으로 기술적 해자와 비즈니스 가치를 동시 어필.
